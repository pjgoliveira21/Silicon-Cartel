# Silicon Cartel

Silicon Cartel is a modular, first-person cooperative simulation project. It integrates hardware assembly, electrical load balancing, and neural network training within a systemic contraband economy. Developed in Unity, the project features a low-poly aesthetic and challenges 1-4 users to manage technical logistics under evolving environmental risks.

---

## 🛠️ Project Overview

**Silicon Cartel** is a systemic simulation where players manage a clandestine computing operation in a world where high-end hardware is a regulated contraband. The project focuses on the intersection of technical logistics, resource management, and risk mitigation.

### Key Technical Pillars
* **Modular Hardware Assembly**: A systemic approach to PC building, focusing on component compatibility and hardware lifecycle management [cite: nomoreram.md].
* **Electrical & Thermal Simulation**: Dynamic load balancing where power consumption and heat generation act as the primary constraints on progression [cite: nomoreram.md].
* **Neural Network Training**: A secondary gameplay layer involving the production of "Neural Models" using accumulated hardware assets [cite: nomoreram.md].
* **Environmental Risk (Heat System)**: A non-linear difficulty curve driven by the player's thermal and electrical signature, affecting world-state and NPC interactions [cite: nomoreram.md].

## ⚙️ Core Systems

### 1. Hardware Lifecycle [cite: nomoreram.md]
* **Acquisition**: Sourcing components through scrap, corporate auctions, or black-market drops.
* **Maintenance**: Technical refurbishment including cleaning, thermal paste application, and component repair.
* **Optimization**: Tuning systems for maximum processing output versus power stability.

### 2. The "Neural" Economy [cite: nomoreram.md]
* Transition from a hardware-flip model to a data-production model.
* Training prohibited AI personalities and cognitive upgrades within private server racks.

### 3. Systemic Risk [cite: nomoreram.md]
* Implicit suspicion mechanics based on visual and auditory cues in the environment.
* Social deduction elements when interacting with NPCs to identify law enforcement infiltrados.
